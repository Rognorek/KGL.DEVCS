Функция: подсчитать количество уникальных значений
	Предыстория: имеется отсортированный вектор целочисленных значений по возрастанию
	Дано программа просит ввести размер вектора
		Сценарий: проверка введеного значения
			Когда значение равно 1
			То выводим 1
			Когда значение 0
			То выводим сообщение о неверном вводе
			Когда значение отрицательное
			То выводим сообщение о неверном вводе
			Когда значение больше либо равно 2
			То вводим значения вектора			
				Сценарий: Осуществить ввод значений вектора
					Когда вводим нечелочисленное число
					То повторяем сценарий ввода текущего элемента
					Когда вводим целочисленное число
					То осуществляем ввод следующего значения
			
			Пример:
			| Размер | Вектор    | Результат |
			|      3 |   2, 2, 2 |         1 |
			|      1 |           |         1 |
			|     -1 |           |    ошибка |
			|      0 |           |    ошибка |
			|      5 | 1,2,3,5,5 |         4 |
