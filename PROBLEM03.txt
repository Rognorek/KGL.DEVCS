# language: ru
Функция: Подсчет количества уникальных значений в коллекции
    Пользователь вводит данные коллекции: размер коллекции и ее элементы.
    Консоль выводит количество уникальных значений, когда введены верные данные.
    Консоль должна вывести сообщение об ошибке, когда введены неожиданные данные.
    
Предыстория: На ввод поступил отсортированный вектор целочисленных значений по возрастанию

    Сценарий: Печать количества уникальных значений
        Когда <Данные>
        То <Результат>
        
        Пример:
        | Данные                     | Результат |
        | 3 0 1 2                    |         3 |
        | 4 0 1 2 2                  |         3 |
        | 1                          |         1 |
        | 8 -1 -1 -1 -1 -1 0 99 1010 |         4 |

Предыстория: На ввод поступают неожиданные данные

    Сценарий: Ввод неожидаемых данных
        Когда <Данные>
        Тогда <Результат>
        
        Пример:
        | Данные                 | Результат                                          |
        | 0                      | Неверный ввод, ожидается целое положительное число |
        | -1                     | Неверный ввод, ожидается целое положительное число |
        | 1 Ф                    | Неверный ввод, ожидается число                     |
        | 10 1 2 3 4 5 6 7 8 9 Я | Неверный ввод, ожидается число                     |
