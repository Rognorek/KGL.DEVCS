# language: ru

Функция: Работа с очередью FIFO на примере почтового обслуживания.
Пользователь вводит данные симуляции очереди:
время работы офиса (минуты), длина очереди и время обслуживания каждого клиента (минуты).
В результате пользователю выводится количество необслуженных клиентов, на которых не хватило времени.
Когда введены неожидаемые данные - вывести сообщение об ошибке.

Предыстория: Пользователь ввел корректные данные для обработки

    Сценарий: Печать результата обработки
        Когда <Данные>
        То <Результат>
        
        Пример:
        | Данные                               | Результат |
        | 60 10 5 5 10 15 15 9 15 5 8 20       |         3 |
        | 10 10 2 2 2 2 2 2 2 2 2 2            |         5 |
        | 1 1 10                               |         0 |
        | 90 4 1 1 1 1                         |         0 |

Предыстория: Пользователь ввел некорректные данные для обработки

        Сценарий: Вывести сообщение об ошибке
        Когда <Данные>
        То <Результат>
        
        Пример:
        | Данные           | Результат                           |
        | 0                | Ожидается целое положительное число |
        | -1               | Ожидатеся целое положительное число |
        | ""               | Ожидатеся целое положительное число |
        | A                | Ожидатеся целое положительное число |
        | 1 A              | Ожидатеся целое положительное число |
        | 1 10 Я           | Ожидатеся целое положительное число |
        | 20 4 10 100 12 В | Ожидатеся целое положительное число |
