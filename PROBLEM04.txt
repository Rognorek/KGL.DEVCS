# Главная диагональ матрицы — А(i,j), где i==j.

Функция: Определить свойство симметричности квадратной матрицы
	Предыстория: имеется квадратная матрица неизвестных типов данных
	Дано <Матрица> размерностью <Размер>
		Сценарий: ввести размер матрицы
			Когда вводим отрицательное значение или 0
			То выводим сообщение об ошибке
			Когда вводим значение 1
			То выводим "Да"
			Когда вводим значение больше или равно 2
			То Вводим значения матрицы
				Сценарий: ввод значений матрицы
					Когда вводим нечисловое значение
					То повторяем ввод текущего элемента
					Когда вводим числовое значение
					То переходим к вводу следующего элемента
		Сценарий: выводим свойство матрицы
			Когда введенная матрица является симметричной
			То выводим "Да"
			Когда введеная матрица не является симметричной
			То выводим "Нет"
			
			Пример:
			| Размер | Матрица              | Результат      |
			|      2 | 0,1;1,0              |             Да |
			|      2 | 1,1;1,0              |            Нет |
			|     -2 |                      |         ошибка |
			|      0 |                      |         ошибка |
			|      1 |                      |             Да |
			|      3 | 1,2,3;2,0,2;3,2,1    |             Да |
			|      2 | 0,X                  | повторите ввод |
